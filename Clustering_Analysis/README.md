# Clustering Analysis

This folder contains clustering modelling and analysis performed using K-Means for customer segmentation in the Customer Churn Analysis project.

## Contents
- K-Means clustering notebook  
- Elbow method analysis results  
- Clustered dataset  
- Cluster summary file  

## Key Activities
- Determined optimal number of clusters using Elbow Method  
- Trained K-Means clustering model  
- Assigned customers to clusters  
- Analysed and interpreted cluster characteristics  

## Tools & Techniques Used
- Python (Scikit-learn)  
- K-Means Clustering Algorithm  
- Elbow Method for cluster selection  

## Purpose
The purpose of this folder is to demonstrate clustering-based customer segmentation, enabling better understanding of customer groups and supporting churn prediction strategies.
