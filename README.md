# Customer Churn Analysis – Stage 2

This repository contains Stage 2 deliverables for the Customer Churn Analysis project in the telecommunications industry.

## Repository Structure

### Data_Preparation
Contains cleaned and preprocessed datasets, train/test split files, scaled datasets, preprocessing notebook, and scaling documentation.

### Clustering_Analysis
Contains elbow method results, trained K-Means model, clustered dataset, cluster summary, and cluster visualisation outputs.

### Stage2_Report
Contains the final Stage 2 written report.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Scikit-learn
- Matplotlib

## Stage 2 Outcomes
- Dataset cleaned and validated
- Features encoded and scaled
- Train/test split completed
- Elbow method applied
- K-Means clustering performed
- Cluster results interpreted
